Telegram Weather, News and Exchange Bot
Description

A Telegram bot that provides weather updates, news, and exchange rates. Users can save favourite cities and get weather by selecting them.

Commands

/start – Shows welcome message and instructions.

/weather – Prompts user to enter a city with /city <city_name>.

/city <city_name> – Fetches weather for the specified city.

/save – Saves the last city looked up to favourites.

/favourites – Displays saved cities as buttons; clicking a button shows weather.

/news – Prompts user to enter a news topic with /topic <topic>.

/topic <topic> – Fetches top news articles for the topic.

/exchange – Prompts user to enter a currency with /currency <currency>.

/currency <currency> – Returns exchange rate of 1 UAH to the specified currency.
